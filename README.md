# BookAI - Your 24/7 Booking Agent Never Sleeps

A modern, sleek landing page for BookAI - an AI-powered booking agent that automates appointments, reservations, and scheduling.

## 🚀 Features

- **Responsive Design**: Fully responsive across all devices (mobile, tablet, desktop)
- **Interactive Chat Demo**: Live chat widget demonstrating AI booking capabilities
- **Smooth Animations**: CSS animations and scroll effects for enhanced UX
- **Modern Aesthetics**: Clean, professional design with gradient accents
- **SEO Optimized**: Proper meta tags and semantic HTML
- **Performance Optimized**: Fast loading with optimized assets

## 📦 Deployment Options

### Option 1: GitHub Pages (Recommended)

1. **Create a new repository on GitHub**
   ```bash
   # Initialize git in your project folder
   git init
   git add .
   git commit -m "Initial commit - BookAI website"
   ```

2. **Connect to GitHub**
   ```bash
   # Replace with your repository URL
   git remote add origin https://github.com/YOUR_USERNAME/bookai-website.git
   git branch -M main
   git push -u origin main
   ```

3. **Enable GitHub Pages**
   - Go to your repository settings
   - Navigate to "Pages" section
   - Select "main" branch as source
   - Click "Save"
   - Your site will be live at: `https://YOUR_USERNAME.github.io/bookai-website/`

### Option 2: Vercel

1. **Install Vercel CLI** (optional)
   ```bash
   npm install -g vercel
   ```

2. **Deploy**
   - Push code to GitHub
   - Visit [vercel.com](https://vercel.com)
   - Import your GitHub repository
   - Click "Deploy"
   - Your site will be live instantly!

### Option 3: Netlify

1. **Deploy via Netlify**
   - Visit [netlify.com](https://netlify.com)
   - Drag and drop the website folder
   - Or connect your GitHub repository
   - Your site will be live with a custom URL

### Option 4: Traditional Hosting

Upload all files to your web hosting via FTP:
- Upload `index.html`, `styles.css`, `script.js`
- Ensure proper file permissions
- Access via your domain

## 🔧 Customization

### Colors
Edit CSS variables in `styles.css`:
```css
:root {
    --color-bg: #0a0a0a;
    --color-primary: #00ff94;
    --color-accent-cyan: #00d9ff;
    /* ... more variables */
}
```

### Content
- Update text in `index.html`
- Modify pricing plans
- Change hero title and descriptions

### Chat Responses
Edit bot responses in `script.js`:
```javascript
const botResponses = {
    default: ["Your custom response here"],
    // ... more responses
};
```
